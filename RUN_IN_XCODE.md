# Task 4 — Run in Xcode (UIKit in code)

1. Create a new project: `iOS App` (`UIKit`, `Storyboard`, `Swift`).
2. Place project folder near this repository.
3. Copy lab sources into app folder:
   ```bash
   cp -r ../task4/src/* ios_app/
   ```
4. In Xcode:
   - remove autogenerated `ViewController.swift`;
   - ensure files from `src/` are in target membership.
5. Set `Signing & Capabilities` Team and run (`Cmd+R`).
